P2 — High: wrong item, delayed order, refund stuck. Resolve 4–8 hrs.

P3 — Medium: how-to, return request, general query. Resolve 24 hrs.

End-to-end call workflow

How a call flows from the carrier to a closed ticket — all automatic
How it's wired: AstronomiQ connects to your cloud telephony provider (Exotel / Ozonetel / Twilio) over REST + WebSocket APIs. Inbound calls hit a virtual number (DID); the provider fires a webhook to AstronomiQ, which runs the IVR, picks the queue, screen-pops the agent, and streams live audio to Voice AI. Every leg is written to a CDR, the recording is stored, and a ticket is auto-created — synced back to Customer 360.
